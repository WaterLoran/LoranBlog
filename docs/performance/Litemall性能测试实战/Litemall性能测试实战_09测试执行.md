# Litemall性能测试实战_09测试执行

## 基准测试

执行单场景(游客浏览商品),100并发,上升周期为10秒,循环次数为10,即脚本会工作100秒

### 游客浏览商品的步骤

1. 初次进入界面
2. 加载主页面信息
3. 点击查看MUJI制造商
4. 加载MUJI制造商的所有商品
5. 点击查看抱枕
6. 加载抱枕详情
