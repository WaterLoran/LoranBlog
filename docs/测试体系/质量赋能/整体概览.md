## 组织级质量赋能

1. 梳理适应新时代软件质量要求的价值观, 营造全员总是质量和价值交付的组织文化
2. 朝着全流程标准化的方向前行, 降低质量成本
3. 强化质量基础设施建设, 扩大自动化规模, 提升研发效能
4. 人员能力是关键, 加强能力建设不容忽视
