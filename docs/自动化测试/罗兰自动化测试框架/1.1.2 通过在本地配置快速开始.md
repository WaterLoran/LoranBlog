



# 前置条件

已经安装好Pycharm

已经安装好python, 要求版本3.9以上

# 创建文件夹并打开

在 E:\Develop 目录下新建 RuoyiTest 文件夹, 然后鼠标右键 -> "Open Folder as Pycharm Project"

# 使用loran命令来初始化

点击最下方的 terminal , 打开控制台, 或者在其他地方打开控制台, 执行命令

```bash
loran init E:\Develop\RuoyiTest
```
