### **通杀心法：钻石应答模型** 💎
应对任意类型面试官，用**四维穿透法**组织答案：
```mermaid
graph TD
A[技术深度] -->|源码/协议/汇编| B[业务广度]
A -->|性能压测/混沌工程| C[风险韧性]
B -->|ROI/市占率/用户体验| D[未来演进]
C --> D
```
